POST to https://jsonplaceholder.typicode.com/posts

Body

{
    "id": 0,
    "userId": 1,
    "title": "Test Post Title",
    "body": "This is a test post body content"
}

Headers

Accept: */*
Content-Type: application/json

Curl

curl -v -X POST 'https://jsonplaceholder.typicode.com/posts' -H 'Accept: */*' -H 'Content-Type: application/json' -d '{ "id": 0, "userId": 1, "title": "Test Post Title", "body": "This is a test post body content" }'
